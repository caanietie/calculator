Operating The Calculator

This calculator works like a small handheld basic calculator which performs addition, subtraction, division and multiplication operations up to a precision of 13 digits.
Below further explains the other functions incorporated into the calculator:
  • MC ⇛ Memory Clear: clears memory of the calculator.
  • M+ ⇛ Memory Plus: increments the memory by the displayed value.
  • M- ⇛ Memory Minus: decrements the memory by the displayed value.
  • MR ⇛ Memory Recall: displays the value in memory.
    An M at the bottom left of the screen indicates that a value has been stored.
  • AC ⇛ All Clear: resets the calculator.
  • BS ⇛ Backspace: deletes the last number on the display.
  • √x ⇛ Square Root: takes square root of displayed value and display result.
  • +/- ⇛ Plus/Minus: changes the sign of the displayed value.
  • π ⇛ pi: display the value of constant pi.
  • x² ⇛ Square: square the displayed value and display result.
  • •00 ⇛ Rounds the display value to 2 decimal places such as in currency.
  • 0• ⇛ Rounds off the display value to 0 decimal places.
  • % ⇛ Percent: use displayed value to calculate a percentage.

Keyboard Control:
You can use the calculator without a mouse by tabbing among the calculator keys. Press "Enter" when a key is focused. This method can be arduous however, since you must tab through all keys in sequence.
Number Keypad Control:
You can use the calculator with with most Number Pads and Keyboards in the most popular browsers for numbers, and the basic functions of addition, subtraction, multiplication, and division and backspace.
Calculating Percentages
Multiplication and Division will convert the displayed value to a percent in decimal form and the operation is completed when = is pressed. Example: To find 20% of 25
Press 25 x 20 % and the display changes 20% to 0.2
Enter = to complete the calculation 25 x 0.2 = 5.
The answer 5 shows in the display.

Addition or Subtraction will add or subtract the percentage of a value. Example: To add 20% to 25
Enter 25 + 20 % and the display changes to 5 (5 is 20% of 25).
Enter = to complete the calculation 25 + 5 = 30.
The answer 30 shows in the display.
